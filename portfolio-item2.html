TheCALCULATOR

CALCULATOR WEB APP

The Calculator is a responsive, user-first web app built with vanilla JavaScript. It demonstrates core front-end skills including precise DOM manipulation, keyboard accessibility, and careful numeric handling so calculations behave like a real calculator.

The UI supports chained operations, decimal inputs, clear and backspace controls, and keyboard shortcuts. Input is validated to prevent invalid expressions and the display updates in real time to give clear feedback while typing and after evaluations.

This project highlights attention to UX details (keyboard + mouse parity), handling edge-case math operations, and building a polished component without external libraries.

Features

  • Basic operations: Addition, subtraction, multiplication and division with correct operator precedence in chained expressions.
  • Decimal support: Precise decimal input and display, preventing multiple decimal points in a single number.
  • Keyboard support: Use number keys, operator keys (+ - * /), Enter for equals, Backspace to delete and Escape/C to clear.
  • Clear & Backspace: Clear (C) resets the entire expression; Backspace removes the last digit/operator.
  • Error handling: Graceful handling of invalid operations (e.g. divide by zero) with user-friendly messages or display states.
  • Responsive UI: Works on mobile and desktop; buttons and display scale to available space.

Technologies Used

  • HTML5 - Semantic layout and accessible buttons
  • CSS3 - Responsive grid/flex layout and polished styling
  • Vanilla JavaScript (ES6+) - Input parsing, evaluation, and keyboard handling
  • Simple state management - track current input, previous value, and pending operator
  • Deployed on GitHub Pages - quick demo hosting
